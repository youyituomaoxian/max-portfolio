# 马守坤 · 个人作品集

全链路设计师个人网站 — 视觉设计 / AI设计 / 品牌设计。

## 技术栈

React 18 + Vite 5 + Tailwind CSS 3 + GSAP + motion

## 本地运行

```bash
npm install
npm run dev
```

## 构建部署

```bash
npm run build    # 输出到 dist/
```

部署方案详见 `个人站设计步骤.md` 第九步（推荐 Cloudflare Pages 零成本方案）。
